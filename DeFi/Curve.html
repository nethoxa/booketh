Curve

Curve is an exchange liquidity designed for extremely efficient stablecoin trading (kind of Uniswap/Compound). I'm not gonna explain its inner working details, just focus on what is important. If you feel like it, read the papers 1 and 2.

As an example, suppose we have asset A and wish to exchange it for asset D. For this swap to be possible, A and D must meet the following requirements:

  • Must be of different asset classes (USD, EUR, BTC, ETH...)
  • Must be exchangeable for a Synthetic asset within one of Curve’s pools (sUSD, sBTC...)

The swap can be visualized as A -> B -> C ...wait... C -> D:

  1. The initial asset A is exchanged for B, a synth of the same asset class (USD -> sUSD)
  2. B is converted to C, a synth of the same asset class as D
  3. A wait period passes to account for sudden price movements between B and C
  4. Once the wait period has passed, C is exchanged for the desired asset D

    CURVE --5. Convert synth B to B--> CURVE
    CURVE --6. Send B--> YOU
